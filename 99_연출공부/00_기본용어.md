# 카메라 연출 정리본

## I. 카메라 구도 (Camera Angle)

| 구도    | 의미          | 관객 위치  |
| ----- | ----------- | ------ |
| 아이 레벨 | 중립, 일상, 동등함 | 같은 눈높이 |
| 로우 앵글 | 자존심, 버팀, 권력 | 올려다봄   |
| 하이 앵글 | 무력감, 열세     | 내려다봄   |
| 더치 앵글 | 불안, 붕괴      | 흔들린 시점 |


## II. 쇼트 크기 (Shot Size)
| 쇼트     | 감정 거리  | 기능     |
| ------ | ------ | ------ |
| 롱 쇼트   | 멀다     | 고립, 관조 |
| 미디엄 쇼트 | 중간     | 대화, 일상 |
| 클로즈업   | 가깝다    | 감정 강제  |
| ECU    | 매우 가깝다 | 압박, 붕괴 |
| 인서트    | 간접     | 의미 강조  |

## III. 카메라 움직임 (Camera Movement)

| 움직임   | 심리 상태  |
| ----- | ------ |
| 고정 샷  | 억제, 체념 |
| 팬     | 인식 변화  |
| 틸트    | 상태 자각  |
| 달리 인  | 결단     |
| 달리 아웃 | 상실     |
| 트래킹   | 동행     |
| 스테디캠  | 통제된 일상 |
| 핸드헬드  | 붕괴     |
| 크레인   | 관조, 구조 |

## IV. 프레이밍과 공간 연출 (Framing)

| 프레이밍    | 의미        |
| ------- | --------- |
| 문틀      | 고립, 경계    |
| 테이블     | 대치, 거리    |
| 프레임 분할  | 단절, 오해    |
| 부분 프레이밍 | 결핍, 소외    |
| 깊이 구성   | 권력 구조     |
| 프레임 밖   | 보이지 않는 압력 |

## V. 테이크 길이 & 컷 타이밍

| 방식     | 효과      |
| ------ | ------- |
| 롱테이크   | 감정 누적   |
| 빠른 컷   | 압박, 과부하 |
| 감정 전 컷 | 억제      |
| 감정 후 컷 | 여운      |
| 침묵 유지  | 불편함     |
| 노컷     | 날것의 감정  |
